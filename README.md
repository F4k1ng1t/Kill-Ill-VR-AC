# Kill Ill VR

Kill Ill is a psychological horror social deduction game where players must uncover which members of their town are carrying a deadly new disease. The infection spreads rapidly, and the longer the infected remain alive, the wider it reaches. With no cure available, the only option left is to identify and eliminate the infected before it’s too late.
